Built an android application - ARGO SMARTS

ARGO SMARTS is an Android application that functions as a POS system, which allows users to perform transactions from their mobile device with or without Internet access. As an application designed using the native Feitian SDK of the devices on which it is installed, ARGO SMARTS is an elegant, innovative application shaped by the impressive set of tools that it offers.

Tech used :

  • Java, EventBus and RX Java, hellocharts, Junit Test, Jetpack navigation and MVVM Architecture and Feitian SDK
  • Room Database, Shared Preferences, USB storages, FTP
