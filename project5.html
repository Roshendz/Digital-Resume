Built a flutter application - TriaTel

It is possible to estimate and manage the damages of your vehicles as well as schedule appointments, take photos, make video reports, make audio requests, and detect the presence of workers with entry and exit stamping. Send directly everything across to the management department.

Tech used :

  • Flutter, Dart, REST API (Http), Getx (State and Route management)
  • SQLite, Get_storage(Shared Preferences), path
  • Google Firebase, Google Map API
