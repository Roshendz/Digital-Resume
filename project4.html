Built an android application - POS CORIPET

POS CORIPET application for Android lets you manage your coupons through Coripet clients (supermarkets) and print barcodes or basic QRCodes of the points you possess. This app can help you easily register new users and new cards by scanning your barcode.

Tech used :

  • Java, REST API (Volley), hellocharts, Glide, GSON, zxing library, Jetpack navigation and MVVM Architecture and Feitian SDK
  • Room Database, Shared Preferences
